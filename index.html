Some popular programming languages
Pronounced "see plus plus", the language was planned as an improvement on the C programming language, adding features based on object-oriented programming.
Java is a general-purpose programming language with the aim to “write once, run anywhere”. It allows for the development of multi platform applications, as well as back-end implementations.
JavaScript is a language used in the browser that makes HTML pages more dynamic and interactive that can change as developers use it. However, nowadays it's gained far more uses than just web pages.
Python is an open source programming language that was made to be easy-to-read and powerful. Python is a good programming language for beginners. It is a high-level language, which means a programmer can focus on what to do instead of how to do it.
